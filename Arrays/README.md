## Problem Details

### 1. Two Sum

Given an array of integers `nums` and an integer `target`, return indices of the two numbers such that they add up to target.

Example:

Input:
nums = [2,7,11,15]
target = 9

Output:
[0,1]

Approach:
We use a HashMap (dictionary) to store elements and their indices while iterating through the list.

Time Complexity: O(n)  
Space Complexity: O(n)
